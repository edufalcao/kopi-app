## Kopi v1.0.0 - Initial Release

**Kopi 1.0.0** is the first release of Kopi, a free and open-source clipboard manager for macOS. It lives in your menu bar and tracks everything you copy — text and images — with smart search, pinning, and a global hotkey.

---

### Clipboard History
- Monitors the clipboard every 0.5 seconds for new text and image content
- Stores history in SwiftData with automatic 30-day purge (configurable)
- Image deduplication via SHA-256 hashing prevents duplicate entries
- Hybrid image storage: small images (<128KB) stored inline, larger images on filesystem

### Quick Panel
- Toggle with `Ctrl+Space` (customizable) or click the menu bar icon
- Search bar filters text items by content
- Filter chips: All, Text, Images, Pinned
- Keyboard navigation: arrow keys to browse, Enter to paste, Backspace to delete
- Right-click context menu on each item: Paste, Pin/Unpin, Delete
- Click any item to paste it into the active app

### History Window
- Full browsing experience via right-click menu bar > "Open History"
- NavigationSplitView with items grouped by date (Today, Yesterday, etc.)
- Detail pane with full content preview and action buttons (Pin, Copy, Delete)

### Settings
- Customize the global hotkey
- Set auto-purge duration (7, 14, 30, 60, or 90 days)
- Toggle launch at login
- Clear all history with confirmation

### Quality
- 19 unit tests covering ClipboardStore and ImageStorageService
- Non-sandboxed for full clipboard access and global hotkey support
- Prompts for Accessibility permission on first launch (required for paste simulation)

### Install
Download **Kopi-v1.0.0.dmg** below, open it, and drag Kopi to your Applications folder.
