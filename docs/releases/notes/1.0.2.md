## Kopi v1.0.2 - Hover Highlighting and About Links

**Kopi 1.0.2** adds mouse hover highlighting to the quick panel, links to the developer website and GitHub repo in Settings, and a polished menu bar icon.

---

### Quick Panel
- Clipboard items now highlight when hovering with the mouse, matching the existing keyboard arrow-key behavior
- Hover and keyboard selection share the same state, so switching between mouse and keyboard is seamless

### Settings
- The About tab now includes links to [edufalcao.com](https://edufalcao.com) and the [GitHub repository](https://github.com/edufalcao/kopi-app)

### Menu Bar
- The "Open History" context menu item now displays an icon, consistent with the other menu items

### Quality
- Extracted `QuickPanelSelection` model for testable selection logic
- Added 9 unit tests covering hover selection, keyboard navigation, clamping, and hover-to-keyboard transitions

### Install
Download **Kopi-v1.0.2.dmg** below, open it, and drag Kopi to your Applications folder.
