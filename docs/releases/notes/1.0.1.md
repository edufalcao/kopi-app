## Kopi v1.0.1 - Stability and Settings Update

**Kopi 1.0.1** consolidates the full post-1.0.0 stabilization pass with more reliable clipboard handling, consistent search behavior, and a cleaner native macOS Settings experience.

---

### Clipboard History
- Paste-from-history now ignores only Kopi's own pasteboard write instead of skipping the next unrelated clipboard update
- Real clipboard copies made immediately after a Kopi paste are now preserved in history

### Quick Panel
- Search now returns matching image items instead of filtering only by text content
- Image and pinned filters now continue to work when a search query is active
- Pasting an item now dismisses the quick panel immediately after the action is triggered

### Menu Bar
- `Open History` and `Settings...` now stay visible on the first attempt instead of flashing briefly and disappearing on first activation

### History Window
- History search now shares the same text and image matching rules as the quick panel
- Pinned image items remain discoverable when filtering and searching together
- The first explicit `Open History` action now survives the app's initial activation instead of being ordered out immediately

### Settings
- The menu bar `Settings...` action now reuses the app's real SwiftUI Settings scene instead of a plain fallback window
- The native macOS Settings toolbar style with tab icons is restored
- Settings reopening now prefers the app menu callback and only accepts selector fallbacks when a real settings window is actually present
- Settings are reorganized into dedicated `General`, `Shortcuts`, `Storage`, `About`, and `Donate` tabs
- The new `Donate` tab includes a Buy Me a Coffee link to support the project

### Quality
- Added regression tests for pasteboard self-write tracking, shared search/filter behavior, first-activation window hiding, deferred status-menu actions, and native settings scene reopening
- The automated suite now covers 36 app tests across storage, search, clipboard-paste safeguards, and settings-window regressions
